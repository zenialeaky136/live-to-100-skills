# Changelog

## v1.0.0 - 2026-03-25

- Initial public release of `live-to-100`.
- Added structured intake and 12-week longevity planning workflow.
- Added longevity risk score model with sub-scores and key drivers.
- Added supplement safety gate with status classification.
- Added auto weekly/monthly report templates and generator script.
- Added complex reminder timetable support:
  - weekday/weekend profiles
  - fixed-time and interval rules
  - quiet-hours filtering
- Added daily nutrition analysis:
  - nutrition balance score
  - calorie deficit status
  - next-day meal adjustment suggestions
- Added demo pipeline to generate 3 markdown outputs in one command.
